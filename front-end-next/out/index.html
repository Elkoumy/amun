Amun

Anonymize your event log.


Amun offers the following features

Risk Quantification

Amun enables you to estimate the risk correlated with revealing a specific user trace and revealing each activity execution within the trace.

Personalized Differential Privacy

Amun estimates the optimal amount of noise to apply differential privacy for each user trace in your event log.

Risky User Traces Filtration

Amun can filter the risky user traces out of your event log.

Scalable Processing

Amun scales up with large-scale event logs.
